# Jarvis Virtual Assistant

This is my personal AI voice assistant project built using Python. I created it to learn how voice recognition, AI, APIs, and automation work together in a real application.

The assistant can listen to voice commands, answer questions using Google Gemini AI, open websites, play music, and read news from different categories.

## Features

- Voice command recognition
- AI chat using Google Gemini
- News by category
- Opens websites like Google, YouTube, Instagram, LinkedIn, etc.
- Plays songs from my music library
- Voice responses using text-to-speech

## Technologies Used

- Python
- Google Gemini AI
- SpeechRecognition
- GNews API
- gTTS
- Pygame
- Requests
- dotenv

## Why I Built This

I wanted to build a project that combines different Python libraries with AI and voice automation. While building it, I learned how to work with APIs, GitHub, environment variables, speech recognition, and AI integration.

I'm planning to keep improving this project by adding more useful features in the future.

## How to Run

1. Clone the repository.
2. Install the required Python libraries.
3. Create a `.env` file and add your API keys.
4. Run:

```bash
python main.py
```

## Note

API keys are stored in a `.env` file and are not included in this repository for security reasons.
